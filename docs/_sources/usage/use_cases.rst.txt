=========
Use cases
=========

The main reasons to use colortools is to generate lists of colors of various sources in various formats.

* :ref:`cmd1`:
* :ref:`cmd2`:

.. _cmd1:

Create a csv file with all colors
---------------------------------

.. code-block:: bash

    ./colortools.py -tasks:csv

.. _cmd2:

Create css files with all colors
--------------------------------

.. code-block:: bash

    ./colortools.py -tasks:cssfonts,cssbgs
